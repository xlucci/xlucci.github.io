misc scripts

generate_residents.py: a silly script that, given a list of class years, generates a properly-formatted residents.shtml file that includes all of the residents currently on the random-hall-[year] lists, as well as the housemasters and GRTs. To run, navigate to the location where the file is to be generated, copy in generate_residents.py, and do

~$ python generate_residents_shtml.py 2014 2015 2016 2017 2018 > residents.shtml

replacing 2014 2015 2016 2017 2018 with however many years you are trying to generate the list for.

-----------------

generate_img.py: a collection of scripts for generating image shtml files from a folder of images. Basically resizes all images to 1200x900 or 900x1200 depending on orientation, creates thumbnails for all images, and generates a main.shtml file containing all images in file_path. To run, navigate to the location with the folder of images, copy in generate_img.py, and do

~$ python generate_img.py foo/ 'Foo Bar 2015' 2

replacing foo/ with the name of the folder with all the images, 'Foo Bar 2015' with the page title, and 2 with the number of columns you want the images arranged in.
